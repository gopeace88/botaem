공모사업 관리
1. 공모사업등록
화면위치	지방보조금 → 보조사업선정 → 공모사업관리 → 공모사업등록 	선행업무	-
작업시기	수시
후속업무	공모사업등록
① “회계년도”, “대상사업명”, “공모명”, “기간”, “보조사업선정진행상태”의 조회조건을 입력합니다.
② [조회] 버튼을 클릭하여 입력한 조회조건에 해당하는 공모사업 목록을 조회합니다.
③ 조회된 목록을 선택 후 [상세] 버튼을 클릭하거나, [공모명]을 클릭하여 공모사업을 조회할 수 있습니다.
① [신규] 버튼을 클릭하여 공모사업 등록 화면으로 이동합니다.
① 사업정보 Tab을 선택한 후 내용을 작성합니다.
회계연도 선택.
전년도사업연계여부 선택.
공모사업을 진행할 대상 사업을 선택.
관리사업에 기본 등록된 “사업개요”, “사업목적”, “지원내용”, “기대효과“, ”사업기간“을 확인하고 수정 가능.
보조사업지원예산금액을 확인하고 보조사업지원금액을 입력
※ 사업지원금액은 대상사업의 보조사업정보에 등록된 예산집행계획에 [30702]민간경상사업보조, [40202]민간자본사업보조 항목의 금액이 있어야 하며 해당 금액이 총사업비 입력의 최대 가능 금 액입니다.
① 기본정보 Tab을 선택한 후 내용을 작성합니다.
② 기본정보를 작성합니다.
공모명
공모기간
공모게시일
③ 담당자 정보를 작성합니다.
등록하는 보조사업 담당자 이외 담당자로 변경하고자 할 경우 조회 버튼을 클릭하고 사용자정보 를 조회하여 선택 변경 가능
① 선정정보 Tab을 선택한 후 내용을 작성합니다.
② 공모사업의 민간보조사업자 선정과 관련된 선정정보를 입력합니다.
선정수, 선정기준안내, 제외대상, 결과통보방법, 공모사업 추진절차, 선정일자, 통보일자
① 신청정보 Tab을 선택한 후 내용을 작성합니다.
② 신청정보를 입력 합니다.
보조금의 비예치형 신청가능 여부
▸ 비예치형사업신청가능여부를 불가능으로 할 경우 보조사업자는 비예치형으로 선택할 수 없음.
하위 민간보조사업자 신청시 입력 가능한 최대 사업비
자부담 최소 금액
▸ 자부담최소신청가능금액을 입력할 경우 민간보조사업자는 공모사업 신청 시 자부담금액을 자 부담최소신청가능금액 이상을 입력해야 합니다.
※ 비예치형이란?
보조사업의 보조금은 기본적으로 상위사업자의 계좌에서 교부 신청한 금액내에서 집행 요청시 마다 지급하는 것을 예치형이라 하며 이와 상반되게 교부 신청시 신청한 보조금액을 상위 사업자의 관리 계좌에서 하위 사업자의 자부담 계좌로 교부 신청 승인 후 일괄 지급되는 형태를 말합니다.
③ 접수정보를 입력합니다.
접수기간, 접수종료시각, 유의사항, 문의 안내관련 입력
④ 보조사업안내서류내역목록에 안내서류를 추가합니다.
[행추가] 버튼 클릭하여 행을 추가한 후 [파일추가]를 클릭하여 안내서류 파일 추가
⑤ 상세신청정보목록을 확인할 수 있습니다.
⑥ 신청정보간소화 여부를 선택할 수 있습니다.
하위 민간보조사업자의 공모사업 신청시 신청서 간소화를 통해 필수적인 내용만으로 신청할 수 있는 기능입니다.
신청정보간소화를 미사용으로 선택할 경우 민간보조사업자는 사업정보등록 시 입력하는 세부추진 계획, 재원조달계획, 예산집행계획 등을 모두 입력해야 합니다.
① 자격검증 Tab을 선택한 후 내용을 작성합니다.
② 자격검증 및 중복수급불가사업 검증여부를 선택합니다.
반드시 온라인자격검증, 오프라인자격검증 중 선택 필수
중복수급불가사업검증여부는 필요한 경우만 선택
③ 온라인자격검증, 오프라인자격 검증 선택 후 [행추가] 추가 버튼 클릭
온라인자격검증
▸ 온라인자격검증항목선택 팝업 창을 통해 자격검증기관 선택 후 조회. 기관별 자격검증정보목록 중 대상을 [선택] 버튼 클릭
오프라인자격검증
▸ 필수여부 선택하고 검증항목을 입력, 연산자를 선택하고 기준값을 입력. “설명 및 비고” 항목에 해당 설명을 간략하게 입력.
④ 중복수급불가사업정보를 추가합니다.
[행추가] 버튼 클릭
중복수급불가사업선택 팝업에서 [조회] 버튼 클릭하여 사업목록 조회
중복수급불가사업 선택하고 [선택] 버튼 클릭하여 추가
① 제출서류 Tab을 선택한 후 내용을 작성합니다.
② [파일추가] 버튼을 클릭하고 등록한 서류 양식 파일을 등록합니다.
③ [행추가] 버튼을 클릭합니다.
추가할 제출 서류가 있을 경우 추가하며 행 추가후 필수여부, 제출서류명을 입력하고 [파일추가] 버튼을 클릭하여 양식 파일을 등록
④ 임시저장 기능을 사용할 수 있습니다.
공모사업 사업 등록 중간에 임시 저장하는 기능입니다.
⑤ [저장] 버튼을 클릭하고 공모사업 등록 정보를 저장합니다.
저장 후 공모사업등록 메뉴에서 등록한 공모사업을 클릭하여 상세화면으로 이동 후 하단의 [ 모 사업연계기안] 버튼을 클릭하여 연계기안 승인 요청을 작성합니다.
연계기안 승인 후 [공모확정] 버튼을 클릭하여 공모사업 최종 확정 등록합니다.
공모사업신청접수
화면위치	지방보조금 → 보조사업선정 → 공모사업관리 → 공모사업신청접수 	선행업무	공모확정 → 하위 보조사업자 공모 신청
작업시기	하위 보조사업자가 공모신청서를 신청받은 후 	후속업무	공모심사
① 조회하고 싶은 검색조건을 입력합니다.
사업연도, 대상사업명, 진행상태(전체/공고확정/접수진행중/접수완료), 공모명으로 조회 가능.
② [초기화] 버튼을 클릭해 검색조건을 초기화 하고 [조회] 버튼을 클릭하여 검색조건에 맞는 공모를 조회합니다.
③ 공고확정/접수진행중/접수완료 등 검색 조건에 맞는 공모를 확인합니다.
전체신청건수 : 해당 공모에 민간기관(단체)에서 신청한 전체 건수
접수결과
▸ 접수대기중 : 아직 접수가 이루어지지 않은 상태인 건수
▸ 보완요청 : 신청한 민간(단체) 중 보완요청 상태인 건수
▸ 접수 : 신청한 민간(단체) 중에서 접수가 된 건수
▸ 접수제외 : 신청한 민간(단체) 중에서 접수가 거절된 건수
④ 목록에서 접수완료 상태로 만들 공모대상 선택 후 [접수완료] 버튼 클릭 시 접수완료여부 확인 창을 확인할 수 있습니다. 접수건수가 없으면 먼저 ⑦[접수목록] 버튼을 클릭해 민간단체의 모든 신청서 를 접수한 후 [접수완료] 버튼을 클릭해 진행할 수 있습니다.
⑤ [접수완료취소] 버튼을 클릭하면 접수완료 상태를 취소하고 접수대기중 상태로 수정됩니다.
⑥ [접수완료]가 된 공모에 대해 [심사요청] 버튼을 클릭 시 심사요청 상태로 변경되고 공모사업선정심 사메뉴에서 심사를 진행할수 있습니다.
⑦ [접수목록] 버튼을 클릭해 공모신청서 목록을 보여주는 화면으로 이동할 수 있습니다.

④ 입력한 검색조건에 맞는 공모신청 목록을 확인 가능합니다.
⑤ [접수]을 클릭하면 공모신청서 내용이 들어 있는 접수진행 화면으로 이동합니다.
