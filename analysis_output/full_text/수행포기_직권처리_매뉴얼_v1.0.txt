수행전 포기
보조사업자가 사업확정되고 사업수행을 하려는 도중에 (보조금 교부받기 이전) 사업을 포기 하게 되는 경우에 선택
수행전 포기 선택시 이후의 진행절차인 교부, 집행, 정산 절차는 수행 할 수 없게됨. (사업목록에 조회되지 않음)
교부받은 금액이 있는 경우 선택 불가.
수행중 포기
보조사업자가 사업확정되고 사업수행을 하는 도중에 (보조금 교부받은 이후) 사업을 포기 하게 되는 경우에 선택
수행중 포기 선택시 이후의 진행절차인 교부, 집행, 정산 절차는 수행 가능
2	(정산은 반드시 해야함, 이자 및 잔액반환)
- 교부받은 금액이 없는 경우에도 선택 가능.
수행포기 신청 제출
수행포기 사유등록 및 증빙서류 파일첨부 이후 저장 및 제출처리
민간보조사업자의 업무를 대신하는 것 이므로 수행포기 저장한 자료를 지자체 담당자에게 제출 처리 함.
수행포기 접수
직권으로 제출처리한 수행포기 자료를 접수하여 승인 또는 반려처리 하는 화면

수행포기 확정처리된 자료의 표시
사업선정, 공모 관련화면에서는 수행전포기, 수행중포기 자료 모두 조회 ①
수행전 포기자료는 교부, 집행, 정산 관련 화면 에서는 조회되지 않음
수행중 포기 자료는 교부, 집행, 정산 관련 화면 에서 조회되며, 해당 업무 수행가능

자부담 집행비율 ①
자부담 재원의 사업비가 포함된 사업의 경우 자부담 재원의 집행비율이 사업비 비율 (순사업비 비율) 보다 같거나 큰 경우에만 집행마감 가능
집행비율이 작은 경우는 집행자료를 수정해야 하는데 집행자료 수정은 직권으로 처리할 수 있 는 영역이 아니므로 IT취약계층 업무대행으로
처리해야 함.
2) 집행/교부 미처리 현황 ②
집행관련 건수가 남아 있는 경우
IT취약계층 업무대행으로 처리해야 함.
교부진행 건수가 남아 있는 경우
교부신청 건수라면 IT취약계층 업무대행으로 처리해야 하며, 교부결정 관련 건수라면 지자체 담당자가 직접 처리 가능 함.

4	3
예치계좌 이자등록 ①
예치계좌에서 발생한 이자금액은 보조사업자가 알 수 있는 금액이 아니므로 보탬e 시스템에서 집행마감 시점까지의 이자금액을 계산하여 화면에 보여줌
보여지는 금액이 자동으로 등록되지는 않으므로 해당 금액을 재원별 금액으로 구분하여 직접 등록 처리 해야함.
자부담 계좌 이자등록
보조사업자의 자부담 계좌에서 발생한 이자금액 을 등록
[자부담계좌에서 발생한 이자입력] ② 버튼을 클릭 해야만 이자금액 입력칸 ③ 이 활성화 됨
자부담 계좌에서 발생한 이자금액이라고 해서 모두 자부담 재원의 이자가 아닐 수 있으므로 총 발생금액을 등록 후 재원별 금액의 등록시 재원을 구분하여 등록
자부담 재원의 이자금액은 반환대상이 아니므로 반환처리 하지 않음.
