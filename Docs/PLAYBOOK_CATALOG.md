## 구현 우선순위

### Phase 1: 핵심 (집행관리)
1. **exec-tax-invoice** - 전자세금계산서 집행등록 (가장 빈번한 작업)
2. **exec-transfer-list** - 이체요청 목록조회
3. **exec-transfer-execute** - 이체실행

### Phase 2: 기초 (회원/로그인)
4. member-login - 로그인 (완료)
5. member-info-view - 회원정보 조회
6. nav-menu - 메뉴 이동 (완료)

### Phase 3: 정산관리
7. settle-close-exec - 집행마감
8. settle-interest-register - 이자등록
9. settle-report-write - 실적보고서 작성

### Phase 4: 나머지
- 사업선정 (복잡, 빈도 낮음)
- 교부관리 (빈도 낮음)
