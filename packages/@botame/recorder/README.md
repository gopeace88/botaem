# @botame/recorder

Playbook recording engine for 보탬e (admin only).

## Overview
Recording functionality used by botame-admin to capture browser actions.

## Installation
```bash
pnpm add @botame/recorder
```

## Usage
```typescript
import { Recorder, generateSelectors } from '@botame/recorder';
```

## Modules
- `capture` - Recorder class
- `smart-selector` - generateSelectors function
