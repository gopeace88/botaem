## 릴리스 단계

### Step 1: 버전 태그 생성

```bash
# package.json 버전 확인 (1.0.0)
git tag -a v1.0.0 -m "Release v1.0.0"
git push origin v1.0.0
```

- [ ] 태그 생성 완료
- [ ] 태그 푸시 완료

### Step 2: GitHub 릴리스 생성

1. [GitHub Releases](https://github.com/gopeace88/botaem/releases/new) 방문
2. 태그 선택: `v1.0.0`
3. 릴리스 제목: `보탬e 관리자 v1.0.0`
4. 설명: `RELEASE_NOTES_v1.0.0.md` 내용 복사
5. assets에 빌드 파일 업로드:
   - `보탬e-관리자-Setup-1.0.0.exe`
   - `보탬e-관리자-1.0.0.dmg`
   - `보탬e-관리자-1.0.0.AppImage`
   - `보탬e-관리자-1.0.0.deb`

- [ ] GitHub 릴리스 생성 완료
- [ ] 모든 빌드 파일 업로드 완료

### Step 3: 빌드 실행

```bash
# Windows (Windows에서 실행)
cd botame-admin
pnpm run build -- --win --x64

# macOS (macOS에서 실행)
pnpm run build -- --mac --universal

# Linux (Linux에서 실행)
pnpm run build -- --linux appimage deb
```

- [ ] Windows 빌드 완료
- [ ] macOS 빌드 완료
- [ ] Linux 빌드 완료
